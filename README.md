### 源码开发

- 1. 克隆源码

     ``` bash
     git clone -b dev https://github.com/Sogrey/Sogrey.github.io.git
     ```

     克隆`clone`·`dev`分支的源码。

     或者使用`TortoiseGit`:

     ![](https://sogrey.github.io/GithubPagePics/imgs/克隆dev分支.png)
- 2. 初始化 hexo :

     ``` bash
     npm install
     ```
- 3. 剩下的交给 `cmd.bat` 吧(需要全局安装Hexo)。
- 4. 如果本地安装了node.js,推荐使用`npm start`预览和`npm run deploy`发布。
